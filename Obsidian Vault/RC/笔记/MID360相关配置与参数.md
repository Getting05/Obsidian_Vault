以200 Hz的频率推送IMU数据，可以在主机计算机上启用或禁用。IMU数据包含3轴加速度和3轴角速度，其方向与点云坐标相同。IMU芯片在点云坐标中的位置为  
`x=11.0mm`  
`y=23.29mm`  
`z=-44.12mm`

AST_LIO的坐标系以IMU为输出坐标原点，所以在IMU坐标系下**点云原点位置的坐标**：

```text
extrinsic_T: [ -0.011, -0.02329, 0.04412 ]
extrinsic_R: [ 1, 0, 0,
               0, 1, 0,
               0, 0, 1]
```
