
## 🍳 已添加的厨房场景功能

### 1. **完整厨房场景** (`QuickConfigs.kitchen_scene()`)
- **厨房环境路径**：使用 Kitchen_set.usd 和 Kitchen_set_instanced.usd
- **丰富的厨房物品**：包含真实的厨房用具、餐具、容器等
- **厨房专用资产路径**：直接使用 Kitchen_set 中的所有资产
- **优化的机器人参数**：适合厨房环境的速度和控制参数

### 2. **简化厨房场景** (`QuickConfigs.simple_kitchen_scene()`)
- **性能优化**：物品数量较少，运行更流畅
- **核心厨房物品**：只包含最重要的厨房用具

### 3. **厨房环境管理方法**
- [`set_kitchen_environment()`]config.py )：启用/禁用厨房环境
- [`get_kitchen_scene_path()`]config.py )：获取厨房场景路径
- [`add_kitchen_asset()`]config.py )：动态添加厨房物品

## 🚀 如何使用

### 在主程序中使用厨房场景：

修改 [`scripts/ultra_stable_create3.py`]ultra_stable_create3.py ) 中的配置选择：

````python
# 在 main() 函数中选择厨房场景
config = QuickConfigs.kitchen_scene(username)              # 完整厨房场景
# 或者
# config = QuickConfigs.simple_kitchen_scene(username)     # 简化厨房场景
````

### 运行厨房场景：

````bash
cd /home/getting/CleanUp_Bench
./run.sh
````

## 📦 包含的厨房物品

### 小垃圾 (10种)：
- 勺子、叉子、量勺、谷物环、钉子
- 小纸片、皱纸、蜡笔、小贝壳、肥皂海绵

### 大垃圾 (13种)：
- **餐具**：盘子、碗、杯子、量杯
- **厨具**：平底锅、锅、铸铁锅、打蛋器、锅铲、擀面杖
- **容器**：瓶子、罐子、罐头、油瓶、盐瓶
- **其他**：书籍、手巾、熨斗

### 家具 (6种)：
- 折叠桌、厨房桌、椅子、木凳、金属凳

## 🎯 特殊优化

1. **真实资产路径**：所有物品都使用 Kitchen_set 中的真实厨房物品
2. **合适的尺寸**：缩放设置为 0.8，保持厨房物品的真实比例
3. **厨房适配参数**：降低机器人速度，增加导航容差，适合厨房环境
4. **性能选择**：提供完整版和简化版两个选项
